# NeoInstaller
A simple installation script to make your SSH Welcome Page a little bit better.

# Run the installation script
## Ubuntu 17.04+
```bash
bash <(curl -s https://raw.githubusercontent.com/venixdeveloper/neoinstaller/main/install.sh)
```
## Raspbian
```bash
bash < <(curl -s https://raw.githubusercontent.com/venixdeveloper/neoinstaller/main/install.sh)
```
